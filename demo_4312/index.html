Basic Google Maps API and Leaflet.js Demo

This is a very basic demo of how to set up a map with a single marker using Google Maps API v3 and Leaflet.js

Both maps have a single marker with a simple popup, two basemaps with a basemap control and that's it.

Read more about choosing a web mapping framework on the corresponding blog post at Cimbura.com.
