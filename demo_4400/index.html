These demos didn't end up being used in a blog post, it just show how you can use Leaflet.js to display images.

Tiles for levels 0-3 were generated using the Stupid Simple Tile Maker. We use the Leaflet setting maxNativeZoom to let the user zoom in closer and make the browser zoom the level 3 tiles.

This map is styled and has an action so that users can see how map tiles work.
